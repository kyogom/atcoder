# atcoder
Atcoderの回答を載せておくリポジトリ

### 使い方
`atcoder-tools gen {abc056}`  
`cd {$workspace_root}/contests/abc056/A`  
`atcoder-tools test`  

### git rule
issue - 問題単位 or コンテスト単位  
commit - 問題単位


### directory rule

/contest ---- 解いたコンテストの問題  
/examples --- コピペで使える便利なコード  

### naming rule

ファイル名は問題文のURL末尾を使う
